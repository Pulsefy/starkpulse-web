## Getting Started

### Prerequisites

- Node.js 18.0.0 or higher
- npm 
- Git

### Installation

1. Clone the repository:
```bash
git clone https://github.com/Pulsefy/Starkpulse.git
cd starkpulse
```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm run dev
```

4. Access the application at http://localhost:3000
